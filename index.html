HTML TAGS

SI Tag Names Description
01 <p> Defines a paragraph.
02 <b> Defines bold text.
03 <i> Defines a part of text in an alternate voice or mood.
04 <u> Defines text that should be stylistically different from normal text.
05 <br> Defines a single line break.
06 <link> Defines the relationship between a document and an external resource (most used to link to style sheets).
07 <tr> Defines a row in a table.
08 <td> Defines a cell in a table.
09 <table> Defines a table.
10 <th> Defines a header cell in a table.
11 <strong> Defines important text and is typically displayed in bold.
12 <body> Defines the document's body
13 <title> Defines a title for the document.
14 <!DOCTYPE> Defines the document type.
15 <html> Defines the root of an HTML document.
16 <a> Defines a hyperlink.
17 <h1> to <h2> Defines HTML headings.
18 <head> Defines information about the document.
19 <img> Defines an image.
20 <style> Defines style information for a document.
21 <li> Defines a list item.
22 <ol> Defines an ordered list.
23 <ul> Defines an unordered list.
24 <div> Defines a section in a document.
25 <header> Defines a header for a document or section.
26 <footer> Defines a footer for a document or section.
27 <sub> Defines subscripted text.
28 <area> Defines an area inside an image-map.
29 <address> Defines contact information for the author/owner of a document.
30 <center> Defines centered text.
